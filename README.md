# UnitConverterApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.0.8.
The application is a unit conversion toolimplementing the conversion between different units of measurement

## Structure

Directory Structure
app/: Contains the main application module and components.
services/: Contains services like conversion.service.ts for business logic.
category-selector/: Component for selecting measurement categories.
converter/: Component for performing unit conversions.
conversion-result/: Component for displaying conversion results.

## Development server

Run `ng serve` or `npm run` for a dev server. Navigate to `http://localhost:4200/`.
